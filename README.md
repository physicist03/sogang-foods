# 🍽️ 서강대 근처 식당 리스트

서강대학교 주변에서 직접 다녀온 식당들을 **마크다운(md) 파일 하나로 관리**하고,  
이를 **GitHub Pages 웹사이트로 자동 렌더링**하는 개인용 맛집 정리 프로젝트입니다.

> md만 수정하면 사이트가 자동으로 업데이트됩니다.

---

## ✨ 주요 특징

- 📄 **md 기반 데이터 관리**
  - `foods.md` 파일 하나로 식당 정보 관리
  - GitHub에서 바로 수정 가능

- 🕒 **영업 상태 자동 판별**
  - 현재 시간 기준으로  
    **영업 중 / 브레이크타임 / 영업 종료 / 휴무** 표시
  - `12:00–02:00` 같은 **익일 마감 영업**도 지원
  - 24시간 영업 지원 (`매일 00:00-24:00`)

- ⏰ **영업시간 그대로 표시**
  - md에 입력한 영업시간 문구를 카드에 그대로 노출

- 🔍 **검색 & 필터**
  - 이름 / 주소 / 종류 검색
  - 정문 · 남문 · 후문 · 술집 · 기타 구역 필터
  - **영업 중인 가게만 보기** 토글 지원

- 🌐 **지도 연동**
  - 네이버/카카오/구글 지도 링크 바로 열기

- 🎨 **라이트 테마 UI**
  - 카드 기반 레이아웃
  - 모바일/데스크톱 모두 대응
